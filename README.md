# RAMP Predictive Maintenance Assistant

Simple Streamlit chatbot for the capstone project:
**Predictive Maintenance and Decision Support System for Warehouse Automation**

## Files
- `app.py` — main Streamlit app
- `requirements.txt` — dependencies

## Local run
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Needed secret
Set `OPENAI_API_KEY` in Streamlit secrets or as an environment variable.
